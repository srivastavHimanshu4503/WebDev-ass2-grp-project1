Calculating U' matrix

If our U matrix has pivot element other than one than it will not give us the right answer. So, to get the accurate result we need to make the pivot elements one by dividing the entire row by the its respective pivot element and named that matrix as U'.

U' matrix is obtain by dividing the row elements of U matrix by its pivot value i.e., Dividing R1 by a11, R2 by a22 and R3 by a33.

If U matrix's pivot element value is already one then U and U' will be same.

Calculating LDU matrix

LDU decomposition (or LDU factorization) is a form of matrix decomposition used in numerical linear algebra. It involves breaking down a square matrix into the product of three simpler matrices: a lower triangular matrix (L), a diagonal matrix (D), and an upper triangular matrix (U).
